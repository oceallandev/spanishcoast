App Guide

This is the living guide for the Spanish Coast Properties app. It is updated together with the app so it always matches the latest design and functionality.

Search & filter

Find listings fast with filters, cities, and reference search.

Map view

Explore listings geographically with clean, branded markers.

Save favourites

Build a shortlist and keep it synced to your account.

Brochures

Search & Filters

Use the search bar, quick city buttons, and filters to narrow results quickly.

Reference search (SCP-XXXX) Town / area Beds / baths / price Sort: newest first
  • Tip: you can paste a reference like SCP-1898 to open the exact listing.
  • Tip: use Saved mode to show only your shortlist.

Map View

The map is designed to be practical: explore areas, then open a listing for details.

  • Tap a marker to open the listing.
  • On mobile, use the Map toggle to switch between list and map.
  • Markers use icons that reflect the listing type (apartment, villa, commercial, business, etc.).

Listing Details

Each listing opens with a gallery, key stats, description, features, and contact actions.

Gallery

Swipe/tap through images and open full-screen.

Mini map

Quick location context with a Google Maps link when available.

Actions

Save, request a visit, call, copy link, and share.

Original ref (privileged)

Admins/partners can see the source reference; clients only see SCP refs.

Saved Favourites

Save listings to create a shortlist. When you are signed in, your favourites sync to your account.

  • Use Saved to show only favourited listings.
  • As an admin, you can view the favourites inbox to see what users are saving.

Brochures (Print to PDF)

Every listing has a brochure page designed for clean printing to PDF. It also includes share buttons and a white-label mode for collaborators.

Print to PDF WhatsApp link Copy link White-label Area snapshot
  • Open the brochure from the listing and click Print to PDF to save a clean PDF.
  • Use White-label when sharing with partners (removes branding/contact blocks).

Shop (Smart Devices)

The shop showcases products from the WooCommerce store inside the app. Use it for fast browsing, then open the official product page to buy, or request installation and setup.

Auto-synced from WooCommerce Open in shop WhatsApp Install support
  • Use search + category filters to find devices quickly.
  • Open a product and request installation for a clean setup and handover.

Share (WhatsApp / Link / Email)

Sharing is built for real-world usage: fast, clear, and consistent across devices.

  • Copy link is the most reliable option across all phones and browsers.
  • WhatsApp and Telegram are ideal for sending a brochure link to clients instantly.
  • When needed, use Email link for structured follow-ups.

Account (Login, Password, Sessions)

Your account is mainly used to sync favourites across sessions and enable role-based dashboards.

Sign in

Use email + password. You can also request a magic link where available.

Reset password

Use “Reset your password” from the Account page to set a new password.

Reset login

If login is stuck, use “Reset login” to clear cached auth/session data.

Offline cache

“Clear offline cache” forces a clean refresh of cached pages/assets.

Roles & Dashboards

Different roles unlock different tools. If you need access (agent/agency/developer), create an account then ask an admin to assign your role.

Client Collaborator Agent Agency admin Partner Developer Admin

Admin Tools (Admin Only)

Admin tools exist for operations and privacy. Contacts and leads are stored privately (not in the public app) and are only visible to admins.

  • Favourites inbox: view saved listings from all users and export CSV.
  • CRM: view/import contacts and leads (PII), edit fields, export CSV.
  • User roles: assign roles once users have created their own account.

Troubleshooting

Sign-in stuck / timed out

Try “Reset login”, switch network, disable VPN/ad-block, then try again.

Magic link rate limit

Wait and try again later (providers rate-limit email sends). Password login is the fastest fallback.

Data not updating

Use “Clear offline cache” on the Account page to force a clean refresh.

Diagnostics

Open the Account page with ?qa=1 for setup checks.

Need Help?

If you want a faster result, send us the listing reference (SCP-XXXX), your town/area, and your goal (buy, rent, business transfer, management).
